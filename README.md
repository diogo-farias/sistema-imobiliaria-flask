# Sistema Imobiliária

Sistema web desenvolvido em Python utilizando Flask e MySQL para gerenciamento de imóveis e pessoas.

## Funcionalidades

✔ Cadastro de imóveis  
✔ Listagem de imóveis  
✔ Atualização de imóveis  
✔ Exclusão de imóveis  

✔ Cadastro de pessoas  
✔ Listagem de pessoas  
✔ Atualização de pessoas  
✔ Exclusão de pessoas  

## Tecnologias utilizadas

- Python
- Flask
- MySQL
- HTML
- GitHub
